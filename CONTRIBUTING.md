# 贡献指南

感谢你愿意参与 `DragonScope-AShare`。

这个项目目前仍处于早期阶段，最需要的是：

1. 让数据口径更稳定
2. 让复盘结构更清晰
3. 让系统逐步从 Demo 走向可长期使用

## 当前优先欢迎的贡献方向

1. 单日市场统计的完善
2. 非 ST 连板梯队口径优化
3. 主线题材聚合与排序
4. 周期复盘引擎
5. 多日横向对比引擎
6. 文档和示例输出优化

## 推荐贡献流程

1. Fork 仓库
2. 新建功能分支
3. 做一项聚焦的小改动
4. 同步更新必要文档
5. 提交 Pull Request

## Pull Request 建议内容

请尽量写清楚：

1. 这个 PR 解决了什么问题
2. 改了哪些文件
3. 如何验证
4. 有没有口径变化或潜在影响

## 代码与文档原则

1. 指标口径要显式
2. 涨停、跌停、连板等短线指标默认优先使用非 `ST` 口径
3. 避免把复杂逻辑都塞进一个脚本
4. 文档和实现要保持一致
5. 优先做能复用的模块，而不是一次性代码

## 适合新贡献者的任务

1. 为单日复盘增加新的市场指标
2. 为 README 增加示例展示
3. 为连板梯队计算增加测试
4. 增加周期复盘汇总项
5. 增加对比结果导出能力
